- どんなにしょぼくてもいいから収益化までやってみたい

## 注意

## 機能

- ストップウォッチ
- 時計
- タイマー(1 日以内)
- 10 分休憩勉強法に対応できるやつ
- ダークモード
- 目覚まし時計は web アプリに向かないかも
